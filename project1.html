MyLovelyLib

MyLovelyLib - an interactive platform allowing users to explore some new books. Users can register, browse book descriptions, and leave comments on specific books. The platform enables individuals to form personal lists such as 'Read,' 'Planning to Read,' 'Favorites,' and 'Currently Reading.' Additionally, users have the ability to view profiles and book lists of other members, providing the option to add them as friends. This project was written on ASP.NET Core MVC, using Entity Framework Core, MS SQL Server, Identity Framework, HTML/CSS, JavaScript.
