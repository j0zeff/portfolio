Online Grocery Store

Online Grocery Store - this website provides users with the ability to browse a wide range of products organized into categories and search for specific items. Each product has its own page with a detailed description, images, and specifications. Users can add selected items to their cart. Registered users have the ability to manage their profile and create lists for regular purchases. This project was also written on ASP.NET Core MVC, using Entity Framework Core, PostgreSQL, HTML/CSS, JavaScript.
